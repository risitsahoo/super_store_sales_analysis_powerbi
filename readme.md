## ğŸ“Œ Project Overview

This project focuses on performing an end-to-end sales performance analysis for a retail Super Store using Microsoft Power BI. The objective is to convert raw transactional sales data into actionable business insights that support strategic and operational decision-making.

The project demonstrates practical Data Analyst capabilities including data preparation, exploratory data analysis (EDA), KPI definition, data modeling, DAX calculations, and interactive dashboard development, closely aligned with real-world industry requirements.

---

## â— Business Problem Statement

Retail organizations generate large volumes of sales data on a daily basis. Without structured analysis, decision-makers face challenges in:

- Monitoring overall sales and profitability  
- Identifying high-performing and underperforming regions  
- Analysing category-wise and customer-level performance  
- Supporting data-driven strategic decisions  

This project addresses these challenges by delivering a centralized, interactive Power BI dashboard that enables stakeholders to track performance and identify improvement opportunities efficiently.

## ğŸ“‚ Dataset & Preparation

The analysis is based on the Super Store Sales Dataset, which contains historical order-level retail transaction data.
